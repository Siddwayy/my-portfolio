TO-DO LIST APP

Project Overview

TO-DO-LIST-APP is a lightweight task management utility app built in Unreal Engine 5 using UMG. The application provides a simple, intuitive way to organize daily goals with persistent data storage and full task creation and management capabilities.

Features & Contributions

Task Creation

Add new tasks with titles, descriptions, and priority levels. The interface supports quick entry and clear organization from the start.

Task Management

Edit, delete, and mark tasks as complete. Full CRUD-style workflow keeps your list up to date without leaving the app.

Data Persistence

All tasks are saved automatically and persist between sessions using Unreal’s file system, so you can close and resume without losing data.

Organization & UI

Organize tasks by priority and completion status. The UMG-based interface is clean and user-friendly, with custom widget blueprints for task display, input fields, and actions.

Technical Implementation

UMG Widgets

Custom widget blueprints handle task display, input fields, and buttons. Each UI element is built in UMG for rapid iteration and a consistent look. Event bindings connect user actions to game logic without extra glue code.

Data structures

Task information is stored in custom Blueprint data structures (e.g. title, description, priority, completion flag). These structs are used across widgets and save/load logic so the data model stays consistent.

File I/O & persistence

Save/load uses Unreal’s file APIs to write and read task data to disk. On launch, the app loads persisted tasks and rebuilds the UI from that data. All task changes are reflected in the saved state so sessions survive restarts.

Event-driven architecture

The Blueprint event system drives UI interactions and data updates: button clicks, list updates, and save triggers are wired through events. This keeps UI and logic decoupled and makes it easy to add or change behaviors without touching unrelated code.
