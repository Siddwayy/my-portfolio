TO-DO-LIST-APP (Unreal Engine 5)

TO-DO-LIST-APP is a lightweight task management utility app built in Unreal Engine 5 using UMG. The application provides a simple, intuitive way to organize daily goals with persistent data storage and full task creation and management capabilities.

Project Overview
Skills Unreal Engine 5 - UMG (Unreal Motion Graphics) - Blueprints - UI/UX Design - Data Persistence - File I/O - Software Development
Time Period 2024
Engine Unreal Engine 5 with UMG for rapid UI prototyping and development.
Core Features Task Management: Create, edit, delete, and organize tasks with priority levels.
Data Persistence: Save and load task data using Unreal's file system.
User Interface: Clean, intuitive UMG-based interface for seamless task management.
Technical Implementation Built entirely with Blueprints, utilizing UMG widgets for UI elements, and implementing custom save/load systems for persistent data storage.

Key Features

The TO-DO-LIST-APP provides essential task management functionality:

  • Task Creation: Add new tasks with titles, descriptions, and priority levels
  • Task Management: Edit, delete, and mark tasks as complete
  • Data Persistence: All tasks are automatically saved and persist between sessions
  • Intuitive UI: Clean, user-friendly interface built with UMG widgets
  • Organization: Organize tasks by priority and completion status

Technical Implementation

This project demonstrates proficiency in Unreal Engine 5's UI system and data management:

  • UMG Widgets: Custom widget blueprints for task display, input fields, and buttons
  • Data Structures: Custom data structures to store task information
  • File I/O: Implementation of save/load functionality using Unreal's file system
  • Event-Driven Architecture: Blueprint event system for UI interactions and data updates
