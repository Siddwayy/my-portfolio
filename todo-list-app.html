TO-DO-LIST-APP (Unreal Engine 5)

Download App

Windows .exe File

Skills: Unreal Engine - Blueprints & UMG · User Experience (UX) UX Design & Evaluation · Project Management

Time Estimation: 2 Weeks

A lightweight task management utility app designed and built entirely within Unreal Engine 5 using UMG. The application provides a simple, intuitive way to organize daily goals with persistent data storage and full task creation and management capabilities. Features include full task creation/management (add, edit, delete) and completion tracking, all wrapped in a clean, minimalist UI and a modular design built with core UX principles.

Roles and Responsibility

  1. Designed and built a minimalist, responsive UI using UMG with a focus on clarity and ease of use.
  2. Implemented the core task management logic, allowing users to add, edit, and delete tasks.
  3. Developed a completion tracking system to visually separate pending items from completed goals.
  4. Integrated persistent data storage to ensure all tasks and their statuses are saved between sessions.
  5. Applied key UX principles (visibility, feedback, error prevention) to create an intuitive user flow.
  6. Structured the application with a modular design to allow for future scalability and easy maintenance.
