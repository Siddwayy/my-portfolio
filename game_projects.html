Kivi Board Game (Java)

Time Estimation: 3 Months

A turn-based dice strategy game built in Java Swing, where players compete on a 7x7 grid over 10 rounds. Each player rolls up to three times per turn, aiming to match dice patterns like "three pairs" or "sum over 30" to place stones and score points. The game includes AI opponents, dynamic highlighting of valid moves, and a save/load system using Java serialization. Core gameplay logic is handled through a modular design following GRASP principles, ensuring clean architecture and responsive UI performance.

Roles and Responsibility

  1. Designed and developed the entire game from scratch using Java Swing, including UI, game logic, and architecture.
  2. Implemented the core turn-based system, managing player turns, dice rolls, and score calculations across multiple rounds.
  3. Created the AI player logic capable of autonomous decision-making, dice evaluation, and move selection.
  4. Developed a dynamic grid validation system that checks dice combinations and highlights valid moves in real-time.
  5. Integrated save/load functionality using Java object serialization to preserve game state and allow seamless session recovery.
  6. Followed GRASP and OOP design principles to maintain clean, modular, and easily extensible code.

TowerEcho (Godot4)

Time Estimation: 1 Month

TowerEcho is a fast-paced 2D tower defense shooter developed in Godot Engine using GDScript. Players take control of a mouse-aimed turret to defend their base against randomized enemy waves, each with unique pathing, attacks, and escalating difficulty. Designed for quick reflexes and strategic play, every session blends precision aiming, timing, and resource management into a tightly balanced arcade experience.

The game features real-time combat feedback, including dynamic hit effects, screen shake, and stylized destruction particles. Players can pause, save, and seamlessly resume their progress mid-wave — with tower health, score, and enemy states restored exactly where they left off.

Roles and Responsibility

  1. Designed and programmed the complete gameplay loop — from player turret controls to enemy AI and wave progression.
  2. Developed health, ammo, and regeneration systems, integrating animated UI elements for real-time feedback.
  3. Implemented AI pathfinding and attack behavior, including enemy projectiles, collision detection, and on-death effects.
  4. Built a Save & Resume System to serialize and reload tower health, score, enemy positions, and audio state.
  5. Created interactive menus and HUD, including animated transitions, toggleable sound controls, and in-game overlays.
  6. Added audio-visual polish — particle effects, screen shake, floating damage text, and synchronized SFX for immersive combat.
  7. Optimized game performance and code structure for scalability across future levels and enemy types.
