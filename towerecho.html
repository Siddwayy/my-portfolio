TowerEcho (Godot4)

Download Game

Windows .exe File

Skills: Godot Engine - GDScript - 2D Game Development - AI Systems - Game Architecture - UI/UX Design - Save Management - VFX & SFX Integration

Time Estimation: 1 Month

TowerEcho is a fast-paced 2D tower defense shooter developed in Godot Engine using GDScript. Players take control of a mouse-aimed turret to defend their base against randomized enemy waves, each with unique pathing, attacks, and escalating difficulty. Designed for quick reflexes and strategic play, every session blends precision aiming, timing, and resource management into a tightly balanced arcade experience. The game features real-time combat feedback, including dynamic hit effects, screen shake, and stylized destruction particles. Players can pause, save, and seamlessly resume their progress mid-wave — with tower health, score, and enemy states restored exactly where they left off.

Roles and Responsibility

  1. Designed and programmed the complete gameplay loop — from player turret controls to enemy AI and wave progression.
  2. Developed health, ammo, and regeneration systems, integrating animated UI elements for real-time feedback.
  3. Implemented AI pathfinding and attack behavior, including enemy projectiles, collision detection, and on-death effects.
  4. Built a Save & Resume System to serialize and reload tower health, score, enemy positions, and audio state.
  5. Created interactive menus and HUD, including animated transitions, toggleable sound controls, and in-game overlays.
  6. Added audio-visual polish - particle effects, screen shake, floating damage text, and synchronized SFX for immersive combat.
  7. Optimized game performance and code structure for scalability across future levels and enemy types.
