TowerEcho: 2D Tower Defense Game

TowerEcho is a complete 2D tower defense game built from scratch in Godot Engine using GDScript. Players defend their base with a mouse-controlled turret against waves of enemies that follow randomized paths through handcrafted levels. The game emphasizes skill-based aiming, resource management, and satisfying combat feedback through polished VFX, SFX, and UI.

Gameplay & Core Loop

The core experience revolves around precision mouse aiming and strategic resource management:

  • Mouse-Aimed Turret: Hold to fire continuously toward cursor position. Turret smoothly rotates with instant visual feedback via laser beam traces and muzzle flash.
  • Wave Progression: Two distinct levels with 5-8 waves each. Enemy paths randomize at wave start using Godot's Navigation2D system for varied challenge.
  • Score-Driven Systems: Kills generate score points. Every 100 points regenerates 10% health and 20 ammo. Score multipliers increase with combo kills.

Tower Defense Systems

Your turret serves as both weapon and fragile base that must survive:

  • Health System: 100 base HP with animated progress bar (green → yellow → red). Damage causes screen shake and controller rumble.
  • Ammo Management: 150 max ammo with visible counter. Depletes per shot, regenerates via score milestones or pickup pickups.
  • Combat Polish:
    • Floating damage numbers rise from hit location
    • Hit SFX with pitch variation based on damage
    • Enemy bullets create sparks on block/miss
    • Destruction sequence: explosion particles → screen fade to white → game over

Enemy AI & Combat

Eight enemy types with escalating threat levels create tense decision-making:

  • Pathfinding: Enemies spawn at randomized start points and follow curved paths to player base using A* navigation. Reaching goal deducts 25 tower HP.
  • Damage States: Multi-hit health bars (50-200 HP). Red flash + vibration animation on damage. Death triggers particle burst and score popup.
  • Counter-Attack: 40% of enemies fire homing projectiles at turret. Key Innovation: Enemy bullets collide with player bullets mid-air, creating risk/reward shooting lanes.
  • Boss Wave: Final wave features tanky enemy with shield phase, requiring focus fire to break before damage phase.

Wave Management System

# Wave spawning and difficulty scaling
func spawn_wave(wave_number: int):
    var enemy_count = base_enemies + (wave_number * 2)
    var spawn_delay = max(0.5, 2.0 - (wave_number * 0.1))
    
    for i in range(enemy_count):
        var enemy_type = get_random_enemy_type()
        var spawn_point = get_random_spawn_point()
        spawn_enemy(enemy_type, spawn_point)
        await get_tree().create_timer(spawn_delay).timeout

# Save game state to file
func save_game():
    var save_data = {
        "tower_health": tower_health,
        "tower_position": tower_position,
        "tower_rotation": tower_rotation,
        "score": current_score,
        "ammo": current_ammo,
        "combo_multiplier": combo_multiplier,
        "wave_number": current_wave,
        "enemy_positions": get_all_enemy_positions(),
        "enemy_health": get_all_enemy_health(),
        "audio_position": get_audio_position(),
        "ui_state": get_ui_state()
    }
    var file = FileAccess.open("user://savegame.dat", FileAccess.WRITE)
    file.store_string(JSON.stringify(save_data))
    file.close()
